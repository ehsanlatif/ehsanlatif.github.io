Research Projects

SEAL: Simultaneous Exploration And Localization

Multi-Robot Systems

Developed an integrated solution for simultaneous exploration and localization in multi-robot systems. This approach enables robots to navigate GPS-denied environments while simultaneously building maps and maintaining accurate relative positions.

Read Paper GitHub

CQLite: Communication-Efficient Multi-Robot Exploration

Reinforcement Learning

Developed a communication-efficient multi-robot exploration system using coverage-biased distributed Q-Learning. This approach significantly reduces communication overhead while maintaining exploration efficiency in complex environments.

Read Paper GitHub

PhysicsAssistant

Educational Robotics

Developed an LLM-Powered Interactive Learning Robot for Physics Lab Investigations. This system integrates YOLO object detection, speech recognition, and large language models to create an interactive learning experience for students in physics laboratories.

Read Paper GitHub

G-SciEdBERT

Natural Language Processing

Developed a contextualized language model for Science Assessment Tasks in German. This specialized model is fine-tuned for automatic scoring of German-language science education assessments, providing a valuable tool for educators and researchers.

GitHub Hugging Face

DGORL: Distributed Graph Optimization for Localization

Graph Algorithms

Designed a distributed graph optimization-based relative localization system for multi-robot systems. This approach enables accurate positioning without GPS in dynamic environments, using only local communications between robots.

Read Paper GitHub

Fine-tuning LLMs for Automatic Scoring

Educational AI

Implemented specialized fine-tuning techniques on large language models like ChatGPT for the automatic scoring of educational assessments. This work demonstrates significant improvements in scoring accuracy, outperforming traditional approaches.
